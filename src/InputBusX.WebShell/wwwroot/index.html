Scripted Shape

Movimento paramétrico do analógico: círculo, oval, flick. Enquanto o botão de ativação estiver ativo, a forma é aplicada (aditivo ou override).

None = ativado apenas pelo botão acima. RT/LT = também exige o gatilho pressionado.

Tracking Assist

Enquanto você tracka um inimigo, adiciona movimentos orbitais no stick de mira que "magnetizam" o aim assist do jogo ao alvo. Mais movimento = órbita maior = mais aim assist.

Quando ativado, o macro orbita sozinho usando o raio base mesmo sem input. Combine com TriggerSource=None para um assist passivo permanente.

Head Assist

Mini flick no analógico para travar HS quando o inimigo pula. Três presets (curto/médio/longo) escolhidos pela distância estimada em tempo real.

Progressive No-Recoil

Compensação em 3 fases (início/meio/fim) baseada nas munições. Transições usam smoothstep para não "pular" entre fases.

Fase 1 — Início (1º 1/3)
Fase 2 — Meio (2º 1/3)
Fase 3 — Fim (último 1/3)

Y negativo = puxa para baixo (compensa subida do recoil). Comece com -3000/-5000/-7000 e ajuste.

CrowBar — Anti-Recoil Cooperativo

Amplifica o controle manual de recoil do jogador com um valor fixo HTG. Quanto mais você puxa o analógico para baixo, mais a compensação aumenta. Rapido = 40% assist, Padrao = 90%.

CrowBar só ativa quando o gatilho de tiro está pressionado.
